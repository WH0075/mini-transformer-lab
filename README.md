# MiniTransformer-Lab

## 1. Project Introduction

MiniTransformer-Lab is a PyTorch learning project for implementing a tiny Transformer language model from scratch.

The goal of this project is not to train a large language model, but to understand PyTorch training engineering and the core structure of Transformer step by step.
